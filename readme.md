A simple full-stack coffee ordering app demonstrating **Factory** and **Decorator** design patterns.

## Features

- Select base coffee (Espresso, Latte, Cappuccino)
- Add condiments (Milk, Sugar, Whipped Cream, Caramel)
- View final price instantly

## Design Patterns

- **Factory Pattern** for creating base coffee
- **Decorator Pattern** for adding condiments dynamically

## Tech Stack

- .NET 8 Web API
- Angular
- Swagger

## Run

- Backend: cd CoffeeShop.Api && dotnet run (http://localhost:5272)
- Frontend: cd coffee-shop && ng serve (http://localhost:4200)
